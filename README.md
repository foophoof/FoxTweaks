# FoxTweaks

FoxTweaks is a collection of smaller FFXIV plugin features that I've mostly
pulled in from other small plugins (and usually reimplemented along the way),
because I was tired of having 70 plugins installed, which was quite a pain if I
had to start disabling plugins to debug which one was causing a crash.

Because this is so specific to my own use case (I've generally stripped away
any features I did not use, and mostly do configuration hard coded in the
code), I don't ever plan on releasing this as an actual plugin, but I've made
the source code available in case someone finds parts of it useful.
